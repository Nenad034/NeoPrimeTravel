# 🚀 OpenClaw.ai Integration - NeoTravel Automation Hub

Ovaj folder služi za organizaciju i praćenje autonomnih AI agenata baziranih na **OpenClaw.ai**.

## 🏗️ Trenutni Fokus
Razvoj "Scraper" agenata za B2B portale koji nemaju zvaničan API (BigBlue, Filip Travel, WizzAir, itd.).

## 🤖 Aktivni Agenti
1.  **BigBlue Agent (Waitlist)** - Čeka na konfiguraciju Docker-a.
2.  **Filip Travel Monitor (Draft)** - U fazi definisanja zadataka.

## 📋 Brzi koraci za pokretanje:
1.  Osigurajte da je Docker pokrenut.
2.  Idite u folder repo-a (unutar ovog foldera): `cd openclaw`.
3.  Pokrenite `docker-compose up --build`.
