# 3. Search engine

Search engine je najopterećeniji deo sistema.

Podržava:

* destination search
* hotel search
* flight search
* cruise search
* package search

Search pipeline:

```text

↓
Supplier APIs
↓
Normalization
↓
Aggregation
↓
Deduplication
↓
Ranking
↓
Results
```

# 4. Inventory engine

Inventory engine upravlja svim proizvodima.

Proizvodi:

```text
hotels
flights
transfers
activities
cruises
```

Inventory može biti:

* API inventory
* contracted inventory
* cached inventory

---

# 5. Package engine

Package engine generiše kombinacije.

Primer:

```text
flight + hotel
flight + hotel + transfer
flight + hotel + activity
cruise + flight
```

Engine mora optimizovati:

* cenu
* kompatibilnost datuma
* transfer udaljenost
* marginu

---

# 6. Pricing engine

Pricing engine izračunava finalnu cenu.

Elementi cene:

```text
supplier net
+ markup
+ fees
+ taxes
- discounts
```

Podržava:

* B2C pricing
* B2B pricing
* dynamic markup
* promotions

---

# 7. Booking engine

Booking engine upravlja rezervacijama.

Flow:

```text
search
↓
select offer
↓
reprice
↓
enter traveler data
↓
payment
↓
supplier booking
↓
confirmation
```

Booking mora biti:

* transactional
* auditabilan
* rollback-safe

---
